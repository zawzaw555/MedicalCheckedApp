# 健康診断管理アプリ　 (MedicalCheckedApp) - Servlet/JSP　版

## 概要
Java Servlet . JSP . JDBC を使用して開発した健康診断管理アプリケーションです。
ユーザー情報や健康診断データの管理を実装しています。

## 使用技術
-  Java
-  Servlet
-  JSP
-  SQL
-  JDBC
-  HTML
-  CSS
-  Bootstrap
-  Apache Tomcat

##  主な機能
-  ログイン
-  ユーザー登録
-  ユーザーの健康データ登録
-  ユーザー登録データリスト
-  ログアウト

##  実行環境
-  Apache Tomcat - v10.1
-  Java - JavaSE-21(JRE System Library)
-  SQL

##  実行 URL
http://localhost:8080/MedicalCheckedApp
